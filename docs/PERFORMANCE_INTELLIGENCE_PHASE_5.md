# Performance Intelligence Phase 5

Phase 5 adds dedicated KPI decision pages for:

- Cash
- Sales
- Insurance Premiums
- Joinings
- Funerals

Each KPI page shows:

- Total actual vs target
- Target achievement percentage
- Previous month comparison
- Same month last year comparison
- 3-year average comparison
- The selected franchise rank and movement
- A 12-month actual vs target visual
- Top 10 leaderboard
- Needs-attention list

The clean public leaderboard still only shows rank, franchise name and movement. Detailed figures stay inside the KPI and franchise dashboards.

No database migration is required for Phase 5.
