# Performance Graphs and Old Franchises Fix

## What changed

### Old franchise control
- The Admin > Users by Franchise page now auto-checks franchises for recent KPI activity.
- Franchises with no Cash, Sales, Insurance, Joinings or Funerals data in the last 3 months are hidden from the active franchise selector.
- Hidden branches appear in a separate **Old Franchises** tab.
- Admin, Super Admin or Finance Manager can activate an old franchise again.
- Active-only franchise selectors are used when linking users to franchises.

### Performance tab
- The main Performance tab now opens a graph-only dashboard.
- Graphs can show the whole active group or one selected franchise.
- Graphs can compare different month/year selections and 6, 12, 24 or 36-month ranges.
- Available graph views: Actual vs Target, Compare to Last Year, Forecast vs Target and Growth Trend.
- Old/hidden franchises are excluded from all graph calculations.

### Speed improvements
- Franchise Details list now uses local browser tabs and search, so switching between Active Franchises and Old Franchises is instant.
- The Performance tab no longer loads large leaderboard tables by default.
