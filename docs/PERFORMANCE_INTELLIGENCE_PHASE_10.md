# Performance Intelligence Phase 10 - Decision Centre

Phase 10 adds the Head Office Decision Centre.

## What it adds

- One executive action screen at `/performance/decision-centre`
- KPI focus cards for Cash, Sales, Insurance Premiums, Joinings and Funerals
- Urgent action list from the Phase 9 insight engine
- Branch watchlist based on branch health and target risk
- Wins to recognise
- Top 10 overall leaderboard

## Purpose

After each monthly PDF/import cycle, management can open one page and see:

- which franchises need support
- which KPI is below target
- which franchises are improving
- which franchises are falling
- which wins should be recognised
- where to click for detailed KPI/franchise analysis

## Migration

No new database migration is required. Phase 10 is a decision layer that reuses the Phase 1-9 performance engine.
